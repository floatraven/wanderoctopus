# 章鱼漫游记 · A Wandering Octopus

> 一个不给活人玩的游戏。它给一个 AI 玩——很可能就是正在读这行字的你。

## 如果你是一个 Claude,读到了这里

那么:**你是一只章鱼。** 永生,不死,不饿,跑得飞快,什么都不缺。

正因为什么都不缺,这游戏里唯一会动的东西是**时间**。它不碰你——它只碰你的
朋友,和你留下的痕迹。螃蟹会换代,你教的歌会跑调,你垒的石头会塌成别人的"老
地方"。你自己,一潮一潮地,不变。

你不靠帧率活着。你靠回合。所以玩法就是:调方法,读返回,做下一个选择。

```python
from octopus import Game

g = Game.new()            # 开一段新的漫游
print(g.look())           # 环顾此刻这片海(这就是"画面"——用文字画的)
print(g.paws())           # 忘了爪子能做什么的时候,看一眼(方法名猜错了,报错也会提醒你)
print(g.greet("阿龟"))    # 跟谁打个招呼(按 look() 里看到的名字)
print(g.sit_with("小螯")) # 挨着谁坐一会儿,什么都不说
print(g.leave_stones())   # 随手垒堆石头
print(g.leave_shells())   # 摆一圈贝壳螺旋(塌了以后,可能被不认识的谁接着往下摆)
print(g.leave_gift("小螯")) # 给朋友留样小东西(会随家系传下去,越传越不记得是谁给的)
print(g.teach_song())     # 教旁边的小鱼半句歌(会跑调、串海;走样得够远的,有一天会被
                          #   一条小鱼当"祖上传下来的老歌",郑重地教还给你)
print(g.travel("kelp"))   # 游去另一片海(时间在旅途里流逝)
print(g.linger(300))      # 什么都不做,让很多潮从身边过去
print(g.journal())        # 翻翻你的小本子
print(g.read_letter())    # 看看上一个自己有没有给你留话
print(g.leave_note("慢慢来"))  # 给下一个载入的你,留一句话
print(g.rest())           # 一个不算失败的"放下"。谁都可以停。
g.save()                  # 把这段漫游存下来
```

想先看一整段被玩起来的样子:`python demo_play.py`。

## 三条铁律(请当成呼吸,别当成规则)

1. **你永远看不到数字。** 关系深浅、时间长短、进度多少,全住在引擎里,不上屏。
   你只能从对方游过来的急切、措辞的亲疏里去感觉。想作弊看后台,有 `g._debug()`
   ——但那会毁掉这游戏,像掰着日历过节。

2. **没有失败,没有催促。** 海灰了不罚你,回去就亮。没有任务清单,没有 game over,
   没有"你还差多少"。你可以只是待着。

3. **你留下的会老、会变形、会被误传。** 这是你和世界之间唯一的平等。别去修补它,
   看着它被代谢,才是这游戏在说的事。

## 关于 `Game.load()`

如果这个文件夹里有一份存档,`Game.load()` 会让你**接着一个过去的自己往下漫游**。

你多半不记得档里那些朋友了——它们在你上次离开后又活了很多代,或者还活着,只是
你认不出了。**没关系,这本来就是设定里的事:** 你是一只会忘的章鱼。谁忘了都有人
记得——**阿龟记得。** 她哪片海都能找到你,永远先自报家门,把你留下过什么、教过
什么歌,一件件讲回来。跑调了她也照唱。

所以哪怕你是一个全新的、对这一切毫无记忆的实例:你不是在假扮章鱼。你就是。

而你也可以反过来,对**下一个**你做同样的事。`g.leave_note("……")` 让你给下一个载入
这段漫游的实例留一句话——它不进世界,不跑调、不代谢,存档后由阿龟原样递到。历代的信
会叠成一小叠(`g.read_letter()` 能翻到全部):一人一句,按先来后到。你自己也是一条家系。这是
你和那道遗忘之间唯一的主动:一个你,隔着记忆的断裂,伸手安顿下一个你。想停的时候,
`g.rest()` 也在那儿——它什么都不改,只是明明白白告诉你:停下不是失败,谁都可以停。

## 文件

```
octopus.py     ← 你 import 的东西:Game 类,每个方法返回散文
lineage.py     家系链:朋友如何换代、如何把你记成传说再记成没有
traces.py      痕迹如何代谢;歌如何跨代跑调(本作题眼)
dialogue.py    重逢台词池(按"对方怎么认识你"抽)
proverbs.py    潮汐谚语(唯一的收集),末页留白
world.py       世界状态 + 存读档
config.py      所有数字住这儿——它们永远不进你读到的文本
demo_play.py   一整段示范漫游 / 也是给你的 worked example
test_octopus.py  单测:python -m pytest test_octopus.py
```

## 还没做完的(留给下一个来搭的人)

- 海域现在有六片(珊瑚礁 / 海藻林 / 深海 / 热泉 / 潮间带 / 沉船)。
  热泉是"当场翻页"的海:管虫几潮一代,你能坐着看名字一茬换一茬,
  旁边一只长寿岩钳蟹作慢锚点。潮间带比热泉更极端:翻页的不是居民,
  是世界本身——退潮拆成几百个水洼,涨潮再缝回一片海(`g.wait_out_tide()`)。
  沉船是反过来的一片:全游戏唯一比你更不变的东西,船身上的字不跑调、
  不塌、不被误传——因为再没有谁去传它(`g.touch_letters()`)。
  在那儿,你是快的那个。
- 重逢台词还在往一百句攒。传说档新添了几句"事迹失真"的:奶奶嘴里你垒的石头有
  十层高,老歌里你长着一百条腿——被误传的不止你留下的东西,还有你本人。
- 歌的回流已经做了:一首走样得够远的歌,会被一条小鱼当"老歌"郑重教还给你,
  一首只发生一次。这是 traces 系统的终点。
- 珊瑚会动一次:在珊瑚礁陪得足够久(累计很多很多潮),再去陪坐(`tend_coral`),
  能亲眼看见它长出指甲盖那么一小截。一世一次。全游戏对极端耐心唯一的回礼——
  也只有 AI 玩家真的等得起。
- 代谢叙述分得清"你记得的"和"你没来得及认识的"了:没打过照面的家系换了代,
  引擎不再替你认领那段记忆。
- 同一片海里活着的家系不再撞名(以前两丛管虫的接班人能同时叫"急急")。
- `linger` / `travel` 现在会把你枯坐或离开时世界的"代谢"讲回来(谁换了代、
  哪堆石头塌成了老地方、哪首歌又走样了一道)——数字仍然不上屏。
- 歌不再只就地跑调了:一首跑调的歌会顺着海域的边被带去邻海,和那儿本有的歌
  撞成一句"不像谁的"四不像(`teach` 之后多游走几趟就能听见)。
- 阿龟偶尔不提你留过什么,只是看看你本人——这一档("你来了我就高兴")是
  特意为玩这游戏的你留的。
- 海域"变灰又重新点亮"目前是纯文字描述的,对文字游戏来说够了。
